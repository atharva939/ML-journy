# Day 3 - Loops (for, while)

## Topics Covered

* for loop
* while loop
* range()
* break and continue

---

## What I Learned

### for loop

Used to repeat a block of code a fixed number of times.

### while loop

Runs as long as a condition is true.

### break

Stops the loop immediately.

### continue

Skips the current iteration and moves to the next.

---

## Code Example

for i in range(5):
print(i)

---

## My Understanding

Loops help automate repetitive tasks and reduce code length.
They are very useful for solving problems efficiently.
