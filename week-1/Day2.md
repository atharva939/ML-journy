# Day 2 - Control Flow (if-else)

## Topics Covered

* if statement
* if-else
* if-elif-else
* Nested if

---

## What I Learned

### if statement

Used to check a condition and execute code if true.

### if-else

Executes one block if condition is true and another if false.

### if-elif-else

Used when there are multiple conditions.

### Nested if

An if statement inside another if statement.

---

## Code Example

age = 18

if age >= 18:
print("Adult")
else:
print("Minor")

---

## My Understanding

Control flow allows programs to make decisions based on conditions.
It is very important for logic building and problem solving.
